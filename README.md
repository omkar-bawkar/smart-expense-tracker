\# 💸 Smart Expense Tracker



A modern fintech-style expense tracking web app that helps users manage spending based on their salary using intelligent budgeting.



\---



\## 🚀 Live Concept



This app doesn’t just track expenses —  

it \*\*guides users on how to spend smarter\*\* based on their income.



\---



\## ✨ Features



\### 👤 User Features

\- Add \& delete expenses

\- Category-based expense tracking

\- Custom category support

\- Daily spending tracking

\- Salary-based budgeting system

\- Financial plan using \*\*50-30-20 rule\*\*

\- Update salary anytime

\- Smart daily budget suggestions



\---



\### 🧠 Smart Budget System

\- Automatically splits salary into:

&#x20; - \*\*Needs (50%)\*\*

&#x20; - \*\*Wants (30%)\*\*

&#x20; - \*\*Savings (15%)\*\*

&#x20; - \*\*Investment (5%)\*\*

\- Suggests ideal daily spending

\- Warns user before overspending

\- Allows user override (freedom-first design)



\---



\### 📊 Analytics

\- Monthly spending visualization

\- Category distribution (Pie chart)

\- Expense trends (Line chart)

\- Smooth animated charts



\---



\### 🛠 Admin Panel

\- View all users

\- Track user spending

\- View salary data

\- Monitor platform usage
